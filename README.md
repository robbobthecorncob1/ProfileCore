# ProfileCore

A backend web API built with .NET and C#. This API is currently a work in progress and is being actively developed to support a GPA Calculator web application and my personal website.

## Prerequisites

[.NET 10 SDK](https://dotnet.microsoft.com/download) installed on your machine.

## Running the API

1. Clone the repository.

2. Open your terminal and navigate to the `ProfileCore` folder.

3. Run the following command to start the server:

   dotnet run
